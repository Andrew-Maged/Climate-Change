Climate Change: Which African Country To Visit After COVID-19¶

Motivation¶

Nowadays, we are in quarantine because of the COVID-19 Pandemic. Therefore, I was curious to learn how to write my first blog when completing my Data Scientist Nanodegree program with Udacity. As an African-Egyptian, I know that Africa has a lot of beautiful places. But, the common idea is that Africa is always warm. In this blog, I will analyze climate change data and provide you with countries to visit any time of the year in moderate temperatures. If you are excited to know which country to visit in Africa after quarantine, let's continue your reading!

Project Description¶

The goal of the project is to analyze climate data provided by Kaggle that contains monthly average temperature reading for countries. Then, provide you with the countries that have moderate temperature in each month. I focused my work on trying to answer the following questions:

  • What is the average temperature for countries since 1973?
  • How does climate change over the years?
  • Which countries to visit in a specific month?

Data Overview:¶

You can find the Kaggle competition in the following link: https://www.kaggle.com/berkeleyearth/climate-change-earth-surface-temperature-data

  • GlobalLandTemperaturesByCountry.csvâ-âMonthly average temperature per country.

def adding_cal_attr(df, date_time_col_name):
    '''
    This function extracts calendar columns from a date column
    Args:
    df (DataFrame) - Input Dataframe
    date_time_col_name (str) - the name of the datetime column
    
    Retrun:
    The inputed DataFrame with the new calendar columns
    '''

    # first we need to check if the timestamp column is dattime datatype or not, if not then we will cast it
    if df[date_time_col_name].dtype != np.dtype('datetime64[ns]'):    
        df[date_time_col_name] = pd.to_datetime(df[date_time_col_name])

    df['Year']       = df[date_time_col_name].dt.year.astype(np.int16)
    df['Month']      = df[date_time_col_name].dt.month.astype(np.int8)
  
    return df

def add_desc_stat(df, column, groupByColList):
    '''
    This function extracts Mean, Min, Max from a numeric column grouped by a list of other columns
    Args:
    df (DataFrame) - Input Dataframe
    column (str) - the name of the column
    groupByColList (list) - the grouped by column lists
    
    Retrun:
    The aggregated DataFrame with the new columns
    '''
    MeanDF = df.groupby(groupByColList)[column].mean().to_frame().reset_index().rename(columns={column:'Mean'})
    MinDF = df.groupby(groupByColList)[column].min().to_frame().reset_index().rename(columns={column:'Min'})
    MaxDF = df.groupby(groupByColList)[column].max().to_frame().reset_index().rename(columns={column:'Max'})
    
    OutputDF = MeanDF.merge(MinDF,how='left',on=groupByColList)\
                     .merge(MaxDF,how='left',on=groupByColList)
    
    return OutputDF
